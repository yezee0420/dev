# 중복 제거 및 오파싱 수정 로직 (2026-03-08)

## 적용 로직 요약

### 1. 중복 제거
- **같은 source_url**: 풍부한 데이터 가진 1건만 유지, 나머지 삭제
- **주종석/주수영**: 같은 부고(고인 주종석, 장례식장 동일) → 주종석(id=22) 삭제, 주수영(id=23) 유지

### 2. 파서 보강
- **제목**: `[부고] 이름 씨(소속) 별세` 패턴 추가 (윤호상 씨(피학살자유족회...) 별세)
- **본문**: `◇A씨 별세·B 소속 직급 관계상` 복수 부고 형식 → B를 핵심인물로 추출

### 3. parse_obituary 우선순위
- 복수 부고 본문에서 key_person 추출 시 **본문 우선** (제목의 "A 별세" 덮어씀)
