# Business Model

## 한 줄 요약

**Freemium + Switch & Save 제휴**. 모두에게 핵심 기능은 무료, 고급 분석·가족 공유는 유료, 더 싼 대체 요금제 추천·카드 추천으로 제휴 수익.

## 수익 모델 3단계

### 1. 제휴 / Switch & Save (Phase 4)

사용자가 입력한 구독·고정비를 분석해서 **"더 저렴한 대안"을 추천 → 전환 시 수수료**.

| 카테고리 | 제휴 상대 후보 | 제안 예시 |
|---|---|---|
| 통신비 | 알뜰폰 MVNO (KT엠모바일, U+알뜰모바일 등) | "월 2만원 요금제를 1만원 알뜰폰으로" |
| OTT | 가족 요금제 매칭 플랫폼 (피클플러스 등) | "혼자 쓰는 프리미엄 → 가족 결제로 월 3천 원" |
| 카드 연회비 | 카드사 제휴 플랫폼 (카드고릴라, 뱅크샐러드) | "연회비 면제 카드 추천" |
| 대출이자 | 대환대출 플랫폼 (핀다, 카카오페이) | "금리 0.5%p 낮은 상품" |
| 인터넷 | 통신사 제휴 | "결합할인 가능 상품" |

**단가 가정**: 전환 건당 5,000~30,000원. MAU 5,000 × 전환율 1% × 평균 1만 원 = 월 50만 원.

### 2. Freemium 구독 (Phase 4 이후)

| 티어 | 가격 | 기능 |
|---|---|---|
| Free | 0 | 구독 20개 + 고정비 10개까지, 기본 알림, 월/연 총합 |
| Plus | ₩2,900/월 | 무제한 항목, 카테고리 리포트, 연간 PDF, 알림 커스텀 |
| Family | ₩4,900/월 | Plus + 최대 6인 공유, 중복 구독 감지 |

**전환율 가정**: 전체 MAU의 3~5%가 유료 전환 (가계부 앱 업계 평균). ARPU ₩100~150/월.

### 3. B2B2C / 화이트라벨 (장기, Phase 6+)

- 카드사·뱅크샐러드·토스 등 슈퍼앱의 "구독 관리" 탭에 임베드
- 라이선스 + MAU 단가
- PMF 이후에만 검토

## 왜 이 조합인가

- **제휴**는 사용자에게 이득(돈 절약)을 주면서 우리도 수익. 죄책감 없는 수익 구조.
- **Freemium**은 헤비유저(구독 20개+) 타겟. 많이 쓰는 사람이 가장 돈 아까워하는 역설을 활용.
- 광고는 **안 함**. 타겟(돈 관리 민감한 유저)에게 역효과 + 브랜드 훼손.

## 비용 구조 (Phase 1~3 기준)

| 항목 | 월 비용 |
|---|---|
| Vercel (무료 티어) | 0 |
| Supabase (무료 티어, 500MB DB·50k MAU) | 0 |
| 도메인 | ~1,500 (연 2만 원대) |
| 카카오 알림톡 (Phase 3, MAU 200 기준·월 2건씩) | ~4,000 |
| 이메일 (Resend 무료 3k/월) | 0 |
| **소계** | **~5,500원/월** |

→ 월 3만 원 예산 안에 Phase 3까지 충분히 수용 가능.

## 리스크

- **이탈률** — 가계부 계열 앱 공통 난제. 온보딩 직후 가치를 못 느끼면 2주 안에 80% 이탈. → Phase 1 "첫 60초 안에 총합 보여주기"가 이래서 중요.
- **제휴 파트너 확보 지연** — Phase 4 진입 전에 DM·콜드메일로 사전 접촉 시작 필요.
- **카카오 정책 변경** — 알림톡은 카카오 비즈 심사 필요. 사전에 검토.

## 초기 KPI

| 단계 | 지표 | 목표 |
|---|---|---|
| Phase 1 | W4 Retention | ≥ 30% |
| Phase 2 | 온보딩 완료율 | ≥ 60% |
| Phase 3 | 알림 opt-in율 | ≥ 50% |
| Phase 4 | 유료 전환율 | ≥ 2% |
