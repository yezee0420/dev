# S&P 500 투자 시뮬레이터

10년간 S&P 500에 정기적으로 투자했을 때 자산이 얼마나 늘어나는지 시뮬레이션하는 웹 서비스입니다.

## 기능

- **실제 월간 수익률 기반**: 2015~2024년 S&P 500 실제 월간 수익률 데이터 사용
- **파라미터 조정**:
  - 월 투자금액 (원)
  - 초기 시드머니 (원)
  - 투자 시작년도 (2015~2024)
  - 투자 기간 (년)
  - 종목/지수: S&P 500, 공격적/보수적 가정, 커스텀 연평균 수익률
- **캔버스 에디터**: 결과 화면의 텍스트를 **클릭하여 직접 편집** 가능
- **최종본 저장**: 편집 후 PNG 이미지로 다운로드

## 실행 방법

```bash
# index.html을 브라우저에서 직접 열기
open index.html

# 또는 로컬 서버로 실행 (CORS 등)
npx serve .
```

## 사용법

1. 왼쪽 패널에서 월 투자금액, 시드머니, 기간 등을 설정
2. **시뮬레이션 실행** 클릭
3. 캔버스에 표시된 텍스트를 **클릭** → 인라인 입력창에서 수정 → **Enter**로 적용
4. **최종본 이미지 저장**으로 PNG 다운로드

## 데이터 출처

- S&P 500 월간 수익률: StatMuse, YCharts 등 공개 데이터 기반 (2015~2024)
- 2025년 데이터는 추정치 포함

## 주의사항

※ 과거 수익률은 미래 수익을 보장하지 않습니다. 투자 판단은 본인 책임입니다.
