# KPI 추적 가이드

## 목적

AI 활용 성과를 **숫자로 증명**하기 위한 주간 측정 체계.
"감으로 좋아졌다"가 아니라 "리포트 작성 4시간 → 1.5시간, 62% 절감"으로 말한다.

## 측정 주기

- **매주 금요일 오후** CSV 파일에 해당 주차 데이터 기입
- **매월 첫째 주 월요일** 대시보드 생성 후 팀장/유관부서 공유

## KPI 4대 카테고리

### 1. 속도 (리드타임)

| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| 주간 리포트 작성시간 | 착수~최종 검수 완료 시간 | 50% 절감 |
| 이상탐지 대응시간 | 알림 발생~1차 분석 완료 시간 | 60% 절감 |
| 질의 대응시간 | 요청 접수~답변 전달 시간 | 50% 절감 |

### 2. 품질 (정확도)

| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| 리포트 수치 오류 건수 | 검수 시 발견된 숫자 불일치 | 0건 유지 |
| 이상탐지 오탐률 | 오탐 / 전체 탐지 비율 | 10% 이하 |
| SQL 검증 통과율 | 자동 검증 통과 / 전체 생성 SQL | 95% 이상 |

### 3. 비즈니스 (임팩트)

| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| 리스크 조기탐지 건수 | AI 탐지로 사전 대응한 건수 | 주 2건 이상 |
| 캠페인 인사이트 활용 건수 | AI 분석이 실제 의사결정에 쓰인 횟수 | 월 3건 이상 |

### 4. 확산 (팀 채택)

| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| 팀원 템플릿 사용 건수 | 공유 템플릿 사용 로그 | 주 5건 이상 |
| 공유 프롬프트 재사용 건수 | 프롬프트 라이브러리 조회/복사 로그 | 주 10건 이상 |

## 보고 팁

- 절대값보다 **변화율**이 임팩트가 크다 ("1.5시간" 보다 "62% 절감")
- **누적 효과**를 강조한다 ("3개월간 총 120시간 절감 = 전임 분석가 0.7명분")
- 비즈니스 지표와 연결한다 ("조기탐지로 예상 손실 X억원 방지")
