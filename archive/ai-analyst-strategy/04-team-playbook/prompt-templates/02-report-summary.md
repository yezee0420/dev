# 프롬프트 템플릿: 리포트 요약문 생성

> 버전: v1.0 | 용도: 주간/월간 리포트 요약문 초안 작성

## 프롬프트

```
당신은 증권사 데이터분석팀의 시니어 애널리스트입니다.
아래 데이터를 바탕으로 {주간/월간} 리포트 요약문을 작성하세요.

## 보고 대상
- 수신자: {팀장/임원/유관부서}
- 보고 목적: {정기보고/이슈보고/의사결정지원}

## 핵심 지표 데이터
{지표명, 금주/금월 값, 전주/전월 값, 증감률을 표로 제공}

## 주요 이벤트
{해당 기간 중 특이 사항, 시장 이벤트, 정책 변경 등}

## 작성 규칙
1. 첫 문장에 가장 중요한 지표 변동을 서술 (숫자 포함)
2. 전주/전월 대비 증감률을 반드시 표기
3. 긍정적 변화와 주의 필요 사항을 균형있게 서술
4. 마지막에 권고안 3가지를 제시:
   - 보수적: 리스크 최소화 관점
   - 중립적: 현행 유지 + 모니터링
   - 공격적: 기회 포착 관점
5. 전체 분량: 400자 이내
6. 숫자에는 반드시 단위 표기 (%, 억원, 건 등)
7. "~것으로 보입니다" 같은 모호한 표현 대신 "~입니다"로 명확히 서술

## 출력 형식
### 핵심 요약
(2~3문장)

### 주목 포인트
- (2~3개 bullet)

### 권고안
1. [보수] ...
2. [중립] ...
3. [공격] ...
```

## 사용 팁

- 비교 기간을 반드시 명시해야 AI가 증감률을 정확히 계산
- "주요 이벤트"를 넣어주면 맥락 있는 해석이 나온다
- 수신자 직급에 따라 어조가 달라지므로 보고 대상을 명시
