# 업무 AI 적용 우선순위 점수화 가이드

## 목적

현재 담당 업무를 객관적으로 분해하고, AI 적용 시 **가장 빠르게 가시적 성과**를 낼 수 있는 과제를 식별한다.

## 점수화 기준

| 항목 | 점수 범위 | 의미 |
|------|-----------|------|
| **반복성** | 1(일회성) ~ 5(매일 동일) | 같은 구조의 작업이 반복되는 정도 |
| **AI 자동화 가능성** | 1(전문가 판단 필수) ~ 5(템플릿화 가능) | 현재 AI 기술로 자동화할 수 있는 비율 |
| **비즈니스 영향도** | 1(내부 참고) ~ 5(경영 의사결정 직결) | 산출물이 매출/리스크/고객경험에 미치는 영향 |

## 우선순위 점수 계산

```
우선순위점수 = 반복성 × AI자동화가능성 × 비즈니스영향도
```

- **최대 125점**, 60점 이상이면 즉시 착수 권장.
- 동점이면 **현재 소요시간이 큰 업무**를 우선.

## 사용법

1. `task-scoring-template.csv`를 열어 본인 업무로 교체한다.
2. 각 항목별 1~5점을 솔직하게 부여한다 (과대평가 금지).
3. 우선순위점수 상위 3개를 **4주 파일럿 과제**로 선정한다.
4. 팀장에게 "이 3개를 AI로 개선하겠다"고 제안한다.

## 카테고리 분류 기준

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| 탐색 분석 | 가설 기반 데이터 탐색 | 고객세그먼트, 상품 수익성 |
| 리포팅 | 정기/비정기 보고서 작성 | 주간 리포트, 경영진 보고 |
| 모니터링 | 지표 이상 감시 및 알림 | 이상거래, 시장 변동 |
| 질의 대응 | 유관부서 데이터 요청 처리 | 수익률 조회, 캠페인 성과 |
| 모델링 | 예측/분류 모델 구축·운영 | 이탈 예측, 신용 스코어링 |

## 점수화 후 행동 체크리스트

- [ ] 상위 3개 과제 확정
- [ ] 각 과제별 "AI 적용 전/후" 기대 효과 1문장 작성
- [ ] 팀장 보고 일정 확보 (1주 내)
- [ ] 파일럿 기간 설정 (2~4주)
- [ ] 성공 기준 정의 (시간절감 %, 오류율 변화 등)
