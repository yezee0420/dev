# 프롬프트 템플릿: 탐색적 데이터 분석 (EDA)

> 버전: v1.0 | 용도: 새로운 데이터셋을 빠르게 파악할 때

## 프롬프트

```
당신은 증권사 데이터분석팀의 시니어 애널리스트입니다.
아래 데이터에 대해 탐색적 분석을 수행하세요.

## 데이터 개요
- 데이터셋명: {데이터셋 이름}
- 행 수: {행 수}
- 컬럼 수: {컬럼 수}
- 기간: {시작일} ~ {종료일}
- 출처: {데이터 출처}

## 컬럼 정보
{각 컬럼의 이름, 타입, 설명, 결측률을 표로 제공}

## 기초 통계량
{df.describe()의 결과를 붙여넣기}

## 분석 요청
1. 데이터 품질 이슈를 식별하세요 (결측, 이상치, 중복, 타입 불일치)
2. 주요 변수 간 상관관계를 설명하세요
3. 눈에 띄는 패턴이나 트렌드를 3가지 제시하세요
4. 추가 분석이 필요한 가설을 2개 제안하세요

## 출력 형식
- 각 항목을 번호로 구분
- 수치에는 반드시 단위 표기
- 불확실한 해석에는 "추가 확인 필요" 표기
- 전체 300자 이내
```

## 사용 예시

```python
import pandas as pd

df = pd.read_csv("customer_trades.csv")

context = f"""
- 데이터셋명: 고객 거래 내역
- 행 수: {len(df)}
- 컬럼 수: {len(df.columns)}
- 기간: {df['date'].min()} ~ {df['date'].max()}
- 출처: 거래 DB

## 컬럼 정보
{df.dtypes.to_string()}

## 기초 통계량
{df.describe().to_string()}
"""
```

## 주의사항

- 고객식별정보(이름, 계좌번호)는 제거 후 입력
- 100만 행 이상이면 샘플링(10%) 후 분석
- AI 결과는 반드시 실제 데이터로 교차 검증
